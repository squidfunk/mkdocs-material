# 欢迎使用 Material for MkDocs

这是一个基于 MkDocs 的强大文档框架，可以将 Markdown 文档转换为专业的静态网站。

## 主要特性

- **响应式设计**：自动适应桌面、平板、手机
- **高度可定制**：颜色、字体、语言、图标等均可自定义
- **性能优异**：轻量快速，提供优秀的用户体验
- **多语言支持**：支持 60+ 种语言

## 快速开始

1. 安装 Material for MkDocs：`pip install mkdocs-material`
2. 在 `mkdocs.yml` 中配置主题：
   ```yaml
   theme:
     name: material
   ```
3. 编写 Markdown 文档
4. 运行 `mkdocs serve` 预览网站

## 文档结构

本项目的文档位于 `docs/` 目录下，您可以通过编辑相应的 Markdown 文件来修改网站内容。
